까보기

  • 택인택에서의 까보기란, 택 안에 또 다른 택(가격)을 확인하는 일련의 과정입니다.
  • 택인택에서 판매하는 하나의 상품에는 여러가지 택(가격)이 존재하며, 까보기 버튼을 클릭하여 자신이 뽑은 택의 가격을 확인 하실 수 있습니다.
  • 택(가격)구성 확인
    구매하고 싶은 상품페이지의 TAG 구성에서 최고가(인터넷최저가)와 최저가를 확인
  • 까보기 버튼 클릭
    상단의 까보기 버튼을 클릭
  • 행운의 숫자 선택
    자신만의 행운의 숫자를 선택
  • Start
    출발지점으로 최근(직전) 오픈한 다른 회원의 도착지점
  • Input
    출발지점에서 선택한 숫자만큼 이동하는 과정(구매완료, 오픈중 인 택은 건너뜀)
  • Result
    내가 받은 택
  • 내가 받은 택(Result)의 가격(할인율)확인
    - 선택된 택(TAG)의 가격(할인율) 확인
    - 내가 받은 택(TAG)의 번호는 '까보기'를 선택한 다음 회원의 시작번호로 쓰임.
  • TAG 사용하기
    확인한 TAG으로 주문 / 양동 / 한번더 선택
